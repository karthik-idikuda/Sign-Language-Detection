# Hand Sign Language Recognition System

This is an AI-based hand sign language recognition system with a graphical user interface. The system uses computer vision to detect and recognize hand gestures in real-time.

## Requirements

- Python 3.x
- OpenCV
- MediaPipe
- NumPy
- TensorFlow
- Tkinter (usually comes with Python)
- PIL (Python Imaging Library)

## Installation

1. Ensure you have Python installed on your system
2. Install the required packages:
   ```bash
   pip install opencv-python mediapipe numpy tensorflow pillow
   ```

## Usage

1. Run the main script:
   ```bash
   python main.py
   ```
2. A window will open with your camera feed
3. Show your hand gestures in front of the camera
4. The system will detect and display hand landmarks
5. Use the Start/Stop buttons to control recognition

## Features

- Real-time hand detection and tracking
- GUI interface with live camera feed
- Hand landmark visualization
- Start/Stop controls
